## Cómo se diferencian de las guías cortas

<Callout kind="info">
  Las **guías paso a paso** del [Centro de Ayuda](/help-center) son tareas puntuales (crear un embarque, timbrar una factura, registrar un pago).

  Estos **tutoriales** son flujos completos: cubren un rol entero o un proceso de extremo a extremo, encadenando varias tareas en orden lógico.
</Callout>

Para tareas individuales puntuales usa el Centro de Ayuda. Para entender cómo encaja todo el día a día de un rol o proceso, usa esta sección.
